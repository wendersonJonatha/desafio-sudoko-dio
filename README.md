# 🧩 Projeto: Jogo de Sudoku em Java

## 🚀 Bootcamp GFT Start #7 - DIO

Este projeto foi desenvolvido como parte do **bootcamp GFT Start #7 - Java**, promovido pela [Digital Innovation One (DIO)](https://www.dio.me/) em parceria com a **GFT**.

O desafio propõe a criação de um jogo de **Sudoku** funcional no terminal, com foco na aplicação prática dos principais conceitos da **Programação Orientada a Objetos (POO)** e manipulação de estruturas de dados em Java.

---

## 📌 Descrição do Projeto

O objetivo principal é desenvolver uma versão interativa do Sudoku que funcione totalmente no terminal. O jogo permite que o jogador preencha os números no tabuleiro e valide suas jogadas conforme as regras do Sudoku.

Durante o desenvolvimento, foram aplicados conceitos fundamentais como:

- Encapsulamento e reutilização de código
- Validação de regras lógicas
- Manipulação de arrays bidimensionais
- Entrada e saída de dados via `Scanner`
- Estruturação de código limpo e modularizado

---

## 🛠️ Funcionalidades

- ✅ Geração de um tabuleiro com posições pré-definidas e campos editáveis
- ✅ Validação de jogadas (linhas, colunas e subgrids 3x3)
- ✅ Verificação de vitória
- ✅ Interface interativa no terminal
- ✅ Mensagens de erro e sucesso para orientar o jogador

---


---

## 🎯 Objetivos de Aprendizado

- Aplicar **POO** na prática (uso de classes e métodos)
- Utilizar **estruturas de dados** como arrays bidimensionais
- Trabalhar com **entrada de dados** do usuário (`Scanner`)
- Refletir sobre **regras lógicas** e condicionais
- Desenvolver uma aplicação simples, porém funcional e bem estruturada

---

## 💡 Tecnologias Utilizadas

- Java 17
- Terminal/Console
- IDE (opcional para desenvolvimento)

---

## 🧠 Conceitos Reforçados

- Classes e Objetos
- Arrays
- Métodos e atributos
- Validação de dados
- Controle de fluxo: `if`, `for`, `while`
- Entrada e saída no console (`System.in` / `System.out`)
